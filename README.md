### Start server with :

<br>

- Start Redis server :
```ruby
redis-server
```

<br>

- Start Sidekiq ( for async services ) :
```ruby
bundle exec sidekiq
```

<br>

- Start rails server ( boot's puma to start ) : 
```ruby
rails s
```
